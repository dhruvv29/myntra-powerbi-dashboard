# myntra-powerbi-dashboard

## Project Overview
This project presents an interactive Power BI dashboard built using Myntra
product data to analyze pricing, discounts, ratings, and category-wise trends
in the e-commerce retail domain.

The dashboard helps in understanding customer behavior and business insights
through visual analytics.

## Tools & Technologies
- Power BI
- Microsoft Excel / CSV
- Data Modeling
- DAX
- Data Cleaning

## Dashboard Features
- Category-wise product distribution
- Price and discount analysis
- Brand performance comparison
- Rating-based insights
- Interactive filters and slicers

## Dataset
The dataset contains Myntra product information such as:
- Product Name
- Brand
- Category
- Price
- Discount Percentage
- Ratings

## Dashboard Preview
Dashboard screenshots are available in the `images/` folder.

## How to Use
1. Download the repository  
2. Open `Myntra_Dashboard.pbix` in Power BI Desktop  
3. Interact with filters to explore insights  

## Business Insights
- Identified high-discount product categories
- Analyzed pricing strategies across brands
- Observed relationship between ratings and pricing
